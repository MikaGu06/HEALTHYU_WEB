Manual de usuario - Healthy U

Este manual explica el uso básico del sistema de escritorio Healthy U, desarrollado por The Big Bang Coders como parte del Proyecto Integrador. El objetivo es facilitar el registro de pacientes, signos vitales, medicación y centros de salud.

1. Pantalla de inicio 2. Iniciar sesión / Registrar usuario 3. Módulo Pacientes 4. Módulo Signos vitales 5. Medicamentos y recordatorios 6. Centros de salud, farmacias y laboratorios

1. Pantalla de inicio

Al abrir Healthy U se muestra la pantalla de inicio con el logo del sistema, el nombre del proyecto y el botón Acceder. Esta vista es similar al diseño de la página web: se presentan brevemente los objetivos del sistema y el logo principal del proyecto.

Desde esta pantalla puedes acceder rápidamente a la sección de Login para comenzar a trabajar con tus pacientes.

2. Iniciar sesión / Registrar usuario

Healthy U cuenta con dos paneles principales en la sección de acceso: INICIO DE SESIÓN y REGISTRAR

  • Inicio de sesión: ingresa tu nombre de usuario y contraseña y haz clic en Login. El sistema valida las credenciales y te permite acceder a los módulos.
  • Registrar: permite crear un nuevo usuario indicando nombre, teléfono y contraseña. El nombre de usuario no es editable una vez creado, para mantener la integridad de los registros.

3. Módulo Pacientes

Una vez iniciado sesión, puedes acceder al módulo Pacientes. Aquí se registran los datos principales de la persona:

  • Nombre completo, CI y fecha de nacimiento.
  • Tipo de sangre y datos de contacto (teléfono, correo).
  • Centro de salud de referencia.

El sistema utiliza estos datos como base para vincular los registros de signos vitales, medicación y centros de salud. Cada paciente puede tener múltiples registros asociados.

4. Módulo Signos vitales

El módulo de Signos vitales permite llevar un seguimiento histórico de los valores medidos a cada paciente. Los campos principales incluyen:

  • Fecha y hora del registro.
  • Ritmo cardíaco (latidos por minuto).
  • Presión arterial sistólica y diastólica.
  • Temperatura corporal.
  • Nivel de oxigenación (SpO₂).

Cada registro queda asociado al paciente seleccionado. De este modo, puedes ver la evolución de los signos vitales a lo largo del tiempo y detectar cambios relevantes.

5. Medicamentos y recordatorios

Healthy U también permite registrar medicamentos y configurar recordatorios de medicación. En este módulo se especifican:

  • Nombre del medicamento y descripción.
  • Dosis y frecuencia recomendada.
  • Horarios en los que el paciente debe tomar el fármaco.

El objetivo es reducir el riesgo de olvidos y mejorar la adherencia al tratamiento. Los recordatorios quedan vinculados al paciente y al medicamento seleccionado.

6. Centros de salud, farmacias y laboratorios

En el módulo de Centros de salud se registran los lugares disponibles para la atención: hospitales, postas, farmacias y laboratorios. Cada centro incluye datos como:

  • Nombre del centro y tipo (hospital, farmacia, laboratorio, etc.).
  • Dirección y teléfono de contacto.
  • Enlace a Google Maps para ubicarlo fácilmente.

Estos datos permiten que el personal de salud tenga siempre a mano las referencias necesarias para derivar pacientes o recomendar lugares de atención.

Conclusión

Healthy U integra en un solo sistema el registro de pacientes, signos vitales, medicación y centros de salud. Este manual sirve como guía rápida para entender el flujo principal del sistema y poder presentarlo correctamente como proyecto integrador.
